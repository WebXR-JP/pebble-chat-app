# PebbleChat アイコンデザイン依頼

## アプリ概要

| 項目 | 内容 |
|------|------|
| アプリ名 | PebbleChat |
| 用途 | VRChat/XRift向けの動画配信アプリ |
| プラットフォーム | macOS / Windows デスクトップアプリ |
| ターゲット | VRChatでイベント配信をしたいユーザー |

## 名前の由来

- **Pebble** = 小石、石ころ
- 既存サービス「TopazChat」（トパーズ＝宝石）に対して、「宝石ではなく石ころ」という謙虚な立ち位置を表現
- TopazChatの負荷分散を目的とした補完ツールという位置づけ

## アイコンの方向性

### コンセプト案

1. **小石 + 配信/ストリーミング**
   - 丸みのある小石のシルエット
   - 電波・ストリーミングを示す要素（波紋、同心円など）

2. **小石 + チャット/コミュニケーション**
   - 吹き出しと組み合わせ
   - VRChat内での視聴体験をイメージ

3. **シンプルな小石**
   - ミニマルでモダンなデザイン
   - グラデーションで立体感

### イメージカラー

- 石っぽいグレー系をベースに
- アクセントカラーは自由（配信っぽい赤/青、XRiftのブランドカラーなど）
- TopazChat（黄色/オレンジ系）とは差別化したい

### 雰囲気

- 親しみやすい、カジュアル
- シンプルで視認性が高い
- macOS/Windows両方に馴染むモダンなデザイン
